# AGECSO Web

Sitio institucional de AGECSO desarrollado en PHP puro con estructura MVC simple.

## Acceso local

```text
http://localhost/AGECSO-web/public/
```

## Objetivo

Reemplazar progresivamente la web actual en WordPress por una solución más liviana y compatible con hosting compartido.

## Relación con rueda de negocios

La plataforma de rueda de negocios se mantiene como proyecto separado. El sitio institucional tendrá un botón hacia esa plataforma cuando se defina la URL final.
